# HaDes-V RISC-V Microcontroller Challenge

Personal implementation of the [HaDes-V](https://www.scheipel.com/hades-v) challenge — building a 32-bit pipelined RISC-V processor in SystemVerilog using a modular, jigsaw-puzzle approach.

## 🎯 Goal
Complete all exercises to earn HaDes-V Bronze, Silver, and Gold certifications through simulation and (optional) hardware workshop.

## 📚 Resources
- [HaDes-V Instruction Guide](https://doi.org/10.3217/nytm4-grv34)
- [Template Repository](https://github.com/tscheipel/HaDes-V/)
- [RISC-V Summit Paper](https://www.researchgate.net/publication/391862217)

## 🛠️ Setup
- Tools: SystemVerilog, Vivado (via provided VM or local install)
- No FPGA required for remote phase

## 🏆 Certification Track
- 🥉 Bronze: Exercises 1–3
- 🥈 Silver: Simulation-to-hardware transfer
- 🥇 Gold: On-site workshop (TBD)
